Detailed Description

Represents a flow of resources or energy between two systems in a simulation model.

The Flow class models the transfer of resources or energy between two systems, designated as the source and destination systems. The name of the flow helps in identifying it within the model. Derived classes must implement the equation method to define the specific calculation of the flow.
