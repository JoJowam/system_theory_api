\doxysubsection{Detailed Description}
Represents a simulation model containing systems and flows. 

The \doxylink{class_model}{Model} class manages collections of \doxylink{class_system}{System} and \doxylink{class_flow}{Flow} objects. It can execute the model over a defined time period, applying the flow equations to the systems at each time step to simulate dynamic behavior. \begin{DoxySeeAlso}{See also}
